## VSCodeとは何か

**Visual Studio Code**（通称：VSCode）は、コードを書くための「エディタ」です。

Microsoftが無料で提供しています。

「エディタ」と聞くとメモ帳やWordを思い浮かべるかもしれません。それに近いのですが、**コードを書くために特化した道具**です。

たとえば、こんなことができます。

- 書いたコードに色がついて読みやすくなる（シンタックスハイライト）
- エラーがある場所を教えてくれる
- コードの補完をしてくれる（途中まで書くと続きを提案してくれる）

Wordで文章を書くのと、メモ帳で文章を書くのは似ているようで全然違います。それと同じで、コードはVSCodeで書くのがずっと楽です。

## インストール方法

**1. 公式サイトを開く**

ブラウザで「VSCode ダウンロード」と検索します。
または `code.visualstudio.com` を開きます。

**2. ダウンロードボタンをクリック**

サイトを開くと、大きな青いボタン（「Download for Mac」または「Download for Windows」）が表示されます。クリックしてダウンロードします。

**3. インストールする**

- **Mac の場合**：ダウンロードされた `.zip` ファイルを開くと「Visual Studio Code」というアプリが出てきます。それを「アプリケーション」フォルダにドラッグします。
- **Windows の場合**：ダウンロードされた `.exe` ファイルをダブルクリックして、指示に従ってインストールします。

**4. 起動する**

インストールできたら、VSCodeを開いてみましょう。
こんな画面が出れば成功です。

---

## 日本語に変える

VSCodeを最初に開くと英語です。日本語に変えられます。

**1. 左側のアイコンから「Extensions（拡張機能）」を開く**

左のバーにブロックを積み重ねたようなアイコンがあります。クリックします。

**2. 検索欄に「Japanese」と入力する**

「Japanese Language Pack for Visual Studio Code」が出てきます。

**3. 「Install」をクリック**

インストール後、「Restart」（再起動）を求められます。クリックして再起動すると日本語になります。

## はじめてのファイルを作る

VSCodeが使えるようになったら、試しにファイルを1つ作ってみましょう。

**1. フォルダを作る**

デスクトップに `practice` という名前のフォルダを作ります。
（デスクトップを右クリック→「新規フォルダ」→名前を `practice` にする）

**2. そのフォルダをVSCodeで開く**

VSCodeのメニューから「ファイル」→「フォルダーを開く」→ `practice` フォルダを選択します。

**3. 新しいファイルを作る**

左側のエクスプローラー（フォルダの中身が表示されているエリア）で、フォルダ名の横にある「新しいファイル」アイコン（紙に＋のアイコン）をクリックします。

`index.html` と入力してEnterキーを押します。

**4. 文字を入力して保存する**

```html
こんにちは
```

と入力して、`Command + S`（Mac）または `Ctrl + S`（Windows）で保存します。

これができれば、VSCodeの基本操作はOKです。

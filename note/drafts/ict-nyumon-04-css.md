## CSSとは何か

CSS（Cascading Style Sheets）は、HTMLで作った骨格に**見た目を指定する**言語です。

「この見出しは青色にする」
「この段落の文字は16pxにする」
「この箱の周りに余白を8px入れる」

こういった指定をHTMLとは別のファイルに書いておきます。

HTMLが「何を置くか」を決めるなら、CSSは「どう見せるか」を決めます。

## CSSを書いてみる

### 手順① CSSファイルを作る

前回作った `html-practice` フォルダをVSCodeで開きます。

新しいファイルを `style.css` という名前で作ります。

### 手順② HTMLにCSSを読み込む

`index.html` の `<head>` の中に1行追加します。

```html
<head>
    <meta charset="UTF-8">
    <title>はじめてのHTML</title>
    <link rel="stylesheet" href="style.css">
</head>
```

`<link rel="stylesheet" href="style.css">` が「このCSSファイルを読み込んで」という指定です。

### 手順③ CSSを書く

`style.css` に以下を貼り付けます。

```css
body {
    font-family: sans-serif;
    color: #333;
}

h1 {
    color: #2d6a9f;
    font-size: 28px;
}

p {
    font-size: 16px;
    line-height: 1.8;
}
```

`Cmd + S` で保存してブラウザをリロードすると、見た目が変わります。

## CSSの基本構造

```css
h1 {
    color: #2d6a9f;
    font-size: 28px;
}
```

CSSは「**セレクタ { プロパティ: 値; }**」の繰り返しです。

| 部分 | 名前 | 意味 |
|---|---|---|
| `h1` | セレクタ | 「どこに」適用するか |
| `color` | プロパティ | 「何を」変えるか |
| `#2d6a9f` | 値 | 「どう」変えるか |

セレクタには、タグ名・class名・id名が使えます。

```css
h1 { }          /* タグ名：h1タグ全部に適用 */
.title { }      /* クラス名：class="title" がついた要素に適用 */
#header { }     /* id名：id="header" がついた要素に適用 */
```

## よく使うプロパティ

```css
/* 文字 */
color: #333;              /* 文字色 */
font-size: 16px;          /* 文字サイズ */
font-weight: bold;        /* 太字 */
line-height: 1.8;         /* 行間 */

/* 背景 */
background-color: #f5f5f5;  /* 背景色 */

/* 余白 */
padding: 16px;            /* 内側の余白 */
margin: 8px;              /* 外側の余白 */

/* 枠線 */
border: 1px solid #ccc;   /* 枠線 */
border-radius: 8px;       /* 角を丸くする */

/* サイズ */
width: 400px;             /* 幅 */
height: 200px;            /* 高さ */
```

## ボックスモデル——paddingとmarginの違い

CSSで一番混乱しやすいのが `padding` と `margin` の違いです。

```
┌──────────────────────────────┐
│          margin（外側の余白）   │
│  ┌────────────────────────┐  │
│  │        border（枠線）    │  │
│  │  ┌──────────────────┐  │  │
│  │  │  padding（内側の余白）│  │  │
│  │  │  ┌────────────┐  │  │  │
│  │  │  │  content   │  │  │  │
│  │  │  │（テキストや画像）│  │  │  │
│  │  │  └────────────┘  │  │  │
│  │  └──────────────────┘  │  │
│  └────────────────────────┘  │
└──────────────────────────────┘
```

- `padding`：枠の**内側**の余白（背景色が入る範囲が広がる）
- `margin`：枠の**外側**の余白（隣の要素との距離）

看護で言えば。

`padding` = 患者さんとベッド柵の間隔（箱の内側）
`margin` = ベッドとベッドの間隔（箱と箱の外側）

「なんかずれる」「余白が思ったより多い」というときは、たいていpaddingとmarginの区別ができていないことが原因です。

### 実際に試す

`style.css` にこれを追加してみましょう。

```css
.box {
    background-color: #e8f4fd;
    border: 2px solid #2d6a9f;
    padding: 20px;
    margin: 30px;
    border-radius: 8px;
}
```

`index.html` の `<body>` に以下を追加します。

```html
<div class="box">
    <p>これはboxクラスの中にあるテキストです。</p>
</div>
```

保存してリロードすると、枠がついた箱が表示されます。

`padding` の値を変えてみる、`margin` の値を変えてみる。数字を変えるたびにリロードして確認すると、違いがつかめます。

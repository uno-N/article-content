## HTMLとは何か

HTML（HyperText Markup Language）は、Webページの**骨格**を作る言語です。

Webページは大きく3つの役割に分かれています。

| 役割 | 言語 | 例え |
|---|---|---|
| 骨格（何が書いてあるか） | **HTML** | カルテの構成・項目 |
| 見た目（どう見えるか） | CSS | 書式・フォント・色 |
| 動き（どう動くか） | JavaScript | インタラクション |

この記事ではHTMLだけを扱います。

## HTMLファイルを作って表示する

### 手順① フォルダを作る

デスクトップに `html-practice` という名前のフォルダを作ります。

（デスクトップを右クリック → 「新規フォルダ」 → `html-practice` と入力）

### 手順② VSCodeでフォルダを開く

VSCodeのメニュー：「ファイル」→「フォルダーを開く」→ `html-practice` を選択。

### 手順③ HTMLファイルを作る

左側のエクスプローラーで「新しいファイル」アイコンをクリックし、`index.html` と入力してEnterを押します。

### 手順④ 以下を貼り付けて保存する

```html
<!DOCTYPE html>
<html>
<head>
    <meta charset="UTF-8">
    <title>はじめてのHTML</title>
</head>
<body>
    <h1>こんにちは</h1>
    <p>はじめてのHTMLページです。</p>
</body>
</html>
```

`Cmd + S`（Mac）または `Ctrl + S`（Windows）で保存します。

### 手順⑤ ブラウザで開く

左側のエクスプローラーで `index.html` を右クリック → 「Finderで表示」（Mac）または「エクスプローラーで表示」（Windows）→ ファイルをダブルクリック。

ブラウザで「こんにちは」と表示されれば成功です。

## HTMLの構造を読む

貼り付けたコードを上から読んでいきます。

### `<!DOCTYPE html>`

「これはHTMLファイルです」という宣言。先頭に書くルールです。

### `<html>` 〜 `</html>`

全体を包む箱。HTMLの内容はすべてこの中に書きます。

### `<head>` 〜 `</head>`

ページの設定情報。**ブラウザの画面には表示されません。**

```html
<meta charset="UTF-8">    ← 文字化け防止の設定
<title>はじめてのHTML</title>   ← タブに表示されるタイトル
```

`<title>` に書いた内容はブラウザの画面ではなく、タブに表示されます。確認してみてください。

### `<body>` 〜 `</body>`

実際に画面に表示される内容を書く場所。

```html
<h1>こんにちは</h1>       ← 大見出し
<p>はじめてのHTMLページです。</p>  ← 段落（テキスト）
```

## 「タグ」の仕組み

HTMLは `<>` と `</>` で囲んで意味を指定します。これを**タグ**といいます。

```html
<h1>見出し</h1>
↑ 開始タグ  ↑ 終了タグ
```

よく使うタグ一覧：

| タグ | 意味 | 使い方 |
|---|---|---|
| `<h1>` 〜 `<h6>` | 見出し（h1が最大） | `<h1>大見出し</h1>` |
| `<p>` | 段落・テキスト | `<p>文章</p>` |
| `<button>` | ボタン | `<button>クリック</button>` |
| `<div>` | グループ化する箱 | `<div>まとめたい要素</div>` |
| `<img>` | 画像 | `<img src="画像のパス">` |

## 試してみる

`<body>` の中を自由に書き換えてみてください。

```html
<body>
    <h1>訪問看護師の学習記録</h1>
    <p>今日からHTMLを始めました。</p>
    <button>クリック！</button>
</body>
```

保存してブラウザをリロード（`Cmd + R` または `F5`）すると変化が確認できます。

ボタンが表示されても、今はまだクリックしても何も起きません。「動き」をつけるにはJavaScriptが必要です。それは次の記事で扱います。

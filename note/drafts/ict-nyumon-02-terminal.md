## ターミナルとは何か

ターミナルは、パソコンに「文字で命令を出す」画面です。

普段、ファイルを開くときはアイコンをダブルクリックしますよね。それは「マウスで操作する」方法です。

ターミナルは同じことを**文字（コマンド）で操作する**方法です。

```
cd Desktop
```

これを入力してEnterを押すと「デスクトップに移動する」という意味になります。

なぜわざわざ文字で操作するのか。理由は2つです。

1. **プログラムのインストールや起動は、コマンドが必要**（マウスでできない操作がある）
2. **何をしたか記録が残る・再現できる**

手術記録を手書きより電子カルテに残す方が検索できて便利、みたいな感覚に近いです。

## ターミナルを開く

**Macの場合**

`Cmd + スペース`でSpotlight検索を開いて「ターミナル」と入力 → Enterで開きます。

または：アプリケーション → ユーティリティ → ターミナル

**Windowsの場合**

スタートメニューで「PowerShell」または「コマンドプロンプト」を検索して開きます。

## VSCodeのターミナルを使う（おすすめ）

実は、VSCodeの中にターミナルが内蔵されています。

VSCodeを開いた状態で：
- `Ctrl + `` `（バッククォート）を押す
- または、メニューの「ターミナル」→「新しいターミナル」

VSCodeのターミナルを使うと、今開いているフォルダの中でコマンドを実行できるので便利です。これからはこちらを使います。

## 最初に覚える3つのコマンド

### `pwd`——今どこにいるか確認する

```bash
pwd
```

`pwd` = Print Working Directory（今いる場所を表示する）

Enterを押すと、今自分がいる場所（フォルダのパス）が表示されます。

```
/Users/yuno/Desktop
```

こんな表示が出れば、今デスクトップにいる、ということです。

迷子になったとき、まず `pwd` を打つ癖をつけると安心です。
病棟の中で「今どこにいるか」確認するのと同じ感覚。

---

### `ls`——今いる場所にある物を見る

```bash
ls
```

`ls` = list（リスト表示）

Enterを押すと、今いるフォルダの中にあるファイルやフォルダの名前が一覧で出ます。

```
Desktop  Documents  Downloads  practice
```

ファイルエクスプローラーで「中身を見る」のと同じ操作をコマンドでやっています。

---

### `cd`——移動する

```bash
cd Desktop
```

`cd` = Change Directory（ディレクトリ＝フォルダを変える）

`cd Desktop` と入力してEnterを押すと、デスクトップに移動します。

移動した後に `pwd` を打つと、確かにデスクトップにいることが確認できます。

**一つ上に戻るには：**

```bash
cd ..
```

`..` は「ひとつ上のフォルダ」という意味です。

---

## 3つのコマンドをまとめて練習

```bash
pwd          # 今どこにいるか確認
ls           # 今いる場所の中身を見る
cd Desktop   # デスクトップに移動
pwd          # 移動できたか確認
ls           # デスクトップにあるものを確認
cd ..        # ひとつ上に戻る
pwd          # 戻れたか確認
```

この流れを1回やってみてください。迷子になったら `pwd` → `ls` で現在地を確認する、この2つが基本です。

---

## まとめ

| コマンド | 意味 | 使うとき |
|---|---|---|
| `pwd` | 今いる場所を確認 | 迷子になったとき |
| `ls` | 今いる場所の中身を表示 | フォルダの中身を確認するとき |
| `cd フォルダ名` | 移動する | 別のフォルダに入るとき |
| `cd ..` | ひとつ上に戻る | 戻りたいとき |

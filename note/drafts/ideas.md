# note記事・投稿アイデアストック

---

## SNS→note誘導パターン（テンプレ）

noteへの流入を作るためのX/Threadsポスト構成。3パターン。

---

### パターンA：「自分の話」型（ストーリーフック）
```
（自分の失敗・迷走・気づきを1〜2文で書く）

詳しくnoteに書きました。
↓
（リンク）
```
**使いどき：** 開発日記・体験記事の告知。感情が入っている記事全般。

---

### パターンB：「問いかけ」型（共感フック）
```
（読者が「あるある」と思う問いや状況を1文で書く）

自分がそうだったから、書きました。
↓
（リンク）
```
**使いどき：** セルフケア・自己理解系の記事告知。

---

### パターンC：「逆説・驚き」型（知的フック）
```
（意外な事実や「え？」となる一文）

なぜかを調べたら、（キーワード）に辿り着いた。
↓
（リンク）
```
**使いどき：** 理論・根拠がある記事の告知。ねぃろの設計思想系。

---

### 注意点
- リンクはスレッドの2投稿目に置く（X）。1投稿目にリンクがあるとリーチが落ちる
- Threadsはリンク直貼りでもリーチは落ちない（仕様が異なる）
- ハッシュタグは#看護師 #個人開発 を基本軸に

## 「言葉にしたら、もうそれは自分じゃない」
**元ネタ：** ACT（ラス・ハリス）読書中の気づき × 音楽制作の自己体験

### 本質
脱フュージョン（思考と自分を切り離す）を理論で説明しようとすると伝わりにくい。
でも、「歌詞を書いたら、もうその気持ちは自分の外にある」という体験は、多くの人がやっている。
感情を言葉にして、メロディに乗せた瞬間、それはもう「自分」ではなく「観察できるもの」になる。
Yunoは10年以上前から、音楽で自然発生的に脱フュージョンをやっていた。

### タイトル案
- 「言葉にしたら、もうそれは自分じゃない」
- 「歌詞を書いてたら、気づいたら楽になってた理由」
- 「音楽が感情を処理できる仕組み、ACTで説明できた」

### メモ
- POPやコミカルなメロディは思考の「深刻さ」を下げる。シリアスなバラードは没入・同一化を強める。ジャンルの「真剣さの文脈」が思考の重みを変えるという仮説。
- てあてチャンネルとも接続できる（音楽作品を題材にした動画で使える切り口）

## 「権力作用を無効化する」
**元ネタ：** 解決できない問題に苦しんでいるときのアプローチとして

### 本質
解決できない問題で苦しんでいるなら、問題そのものより先に、今の自分の中にある権力作用を無効化する。
まず自分がどんな権力作用を持っているかを明確にする。
（「自分はこういう人間だから」「どうせ変われない」「専門家にそう言われた」— 誰の言葉か？）

### X投稿案
```
解決できない問題で苦しんでいるなら、
問題そのものより先に見てほしいことがある。

今のあなたの中にある「権力作用」を無効化すること。

「自分はこういう人間だから」
「どうせ変われない」
「専門家にそう言われた」

それ、誰の言葉ですか？

#看護師 #メンタルケア
```

### Threads投稿案
```
解決できない問題で苦しんでるとき、
問題を解こうとする前にやってほしいことがある。

自分の中にある「権力作用」を見てみること。

誰かに言われた言葉、診断、「あなたはこういう人」って決めつけ。
それが今の苦しさを支えていることがある。

まず、自分がどんな権力作用を持っているか、書き出してみてください。
```

### メモ
「権力作用」という言葉は硬いが、あえてそのまま使うのがYunoらしいかも。「思い込み」「植え付けられた言葉」への言い換えも検討余地あり。

## 「なぜ私はポジティブが苦手だったか」
**元ネタ：** 幸福になりたいなら幸福になろうとしてはいけない（ラス・ハリス）読書中の気づき

### 本質
ポジティブ思考はコントロール戦略の一種。ネガティブな感情をポジティブな考えで上書きして感じないようにする作戦。
それは「本物の感情を偽物と扱う」行為であり、だから空虚でキツい。
ポジティブが苦手だったのは弱いからではなく、偽物だとちゃんと気づいていたから。

### タイトル案
- 「なぜ私はポジティブが苦手だったか」
- 「ポジティブ思考がしんどかった理由、やっとわかった」
- 「『明るく考えよう』が逆にキツかった話」

### note本文下書き
```
「前向きに考えよう」「ポジティブでいよう」

そう言われるたびに、なんか違うと思ってた。
やってみても、空虚で、疲れて、余計しんどくなる。

なんでだろうとずっと思ってた。
最近やっと、言葉にできた。

---

ポジティブ思考って、ある種の「コントロール戦略」だったんだと思う。

ネガティブな気持ちや不安を、
ポジティブな考えで上書きして、感じないようにする作戦。

問題は、それが「本物の感情を偽物と扱う」ということ。

「不安を感じてる自分」を否定して、
「大丈夫と思えてる自分」を演じる。

でも体は知ってる。
これは本当じゃない、と。

だから疲れる。だからキツい。

---

「幸福になりたいなら幸福になろうとしてはいけない」
という本を読んで、これだと思った。

ポジティブでいようとすること自体が、
今ある感情と戦う行為になってしまう。

戦えば戦うほど、感情は強くなる。

---

じゃあどうするか、というのはまた別の話で。
でもまず、「ポジティブが苦手な自分」を責めなくていいと思った。

苦手だったのは、弱いからじゃない。
偽物だと、ちゃんと気づいてたからだ。
```

## 「診断は救いにもなるし、檻にもなる」
**元ネタ：** 物語としてのケア（野口裕二）読書中の気づき

### 本質
専門家によるナラティブは、クライアントをある一定の型にはめてしまう。
でも、クライアント自身がそれを求めている。
「私はこういう人だから」「こういう診断があるからしょうがない」と、今の自分を正当化できるから。
でも、それは問題だと思ってる部分をより強めてしまうだけ。
問題の原因はあってもいい。でも、それが自分の全てじゃない。誰もが持ってるものだ。

### noteタイトル案
- 「診断は救いにも、檻にもなる——『物語としてのケア』を読んで考えたこと」
- 「『自分はこういう人だから』が自分を縛っているかもしれない」
- 「専門家に名前をつけてもらうと、なぜ楽になるのか」

### note構成メモ（PASONA）
- P：診断されて「やっとわかった」と安心したことがある人に向けて
- A：Yunoも看護師として、ラベルが人を楽にする瞬間を見てきた
- S：ナラティブセラピーの「外在化」——問題は自分じゃなく、自分に起きていることとして見る
- O：てあてチャンネルで扱う予定の作品・概念への誘導
- A：「あなたは診断に救われてる？縛られてる？」と問いかけて終わる

### X投稿案
```
「自分はHSPだから」「ADHDだから」

その言葉に救われた人もいる。
でも同時に、その言葉が「だからしょうがない」になってることもある。

診断は、今の自分を正当化するための物語になりやすい。
でも、問題の原因と、あなた自身は別の話だと思う。
```

### Threads投稿案
```
「自分はHSPだから」「ADHDだから」って言葉、
救いになってる人もいれば、そこで止まってる人もいる気がして。

診断ってある種の物語で、専門家から渡されたストーリー。
それが「しょうがない」になった瞬間、少し怖いなと思う。

物語としてのケア、という本を読みながら考えてた。
```

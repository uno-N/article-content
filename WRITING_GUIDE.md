## NoteとZennの使い分け

| | Note | Zenn |
|---|---|---|
| 読者 | 医療者・一般・初心者 | エンジニア・開発者 |
| 内容 | 体験・感情・ストーリー | 手順・コード・技術解説 |
| 文体 | 会話・共感重視 | 構造・正確さ重視 |
| 画像 | スクショ＋感情を表す画像 | 手順説明のスクショ |
| コード | 雰囲気だけ | 詳しく・コードブロック必須 |

同じ開発体験でも両方に書けるネタがある。
**体験・失敗談 → Note、技術手順・解決法 → Zenn** を基本に。
